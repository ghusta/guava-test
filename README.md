# guava-test

Test of the Guava Java library [http://code.google.com/p/guava-libraries/]  
Guava 15 needs Java 6.


## Maven

Dependency :

```xml
  <dependency>
      <groupId>com.google.guava</groupId>
      <artifactId>guava</artifactId>
      <version>15.0</version>
  </dependency>
```
